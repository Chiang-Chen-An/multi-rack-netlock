---
name: controller-skill
description: This skill is to instruct the agents how to write a controller python code for p4 switches.
---

1. The agent should write the controller code in Python, using the appropriate libraries.
2. The agent should ensure that the controller code is modular and follows best practices for Python programming.
3. The agent should include comments in the controller code to explain the functionality of each section and the purpose of each component.
4. The agent should follow the p4-runtime specification for writing the controller code, which can be found at https://p4lang.github.io/p4runtime/spec/v1.5.0/P4Runtime-Spec.html.
