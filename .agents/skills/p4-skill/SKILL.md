---
name: p4-skill
description: This skill is to instruct the agents how to write a p4 code base on the p4 specification.
---

1. The agent should follow the p4 specification to write the p4 code. The p4 specification can be found at https://p4.org/wp-content/uploads/sites/53/2024/10/P4-16-spec-v1.2.5.html.
2. The agent should write the p4 code in a modular way, following the best practices of p4 programming.
3. The agent should ensure that the p4 code is well-documented, with comments explaining the purpose of each section of the code and the functionality of each component.
